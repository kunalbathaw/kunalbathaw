# 🤖 AI Tic Tac Toe

A simple and interactive **Tic Tac Toe game** where you play against an AI powered by the **Minimax Algorithm**. The AI always plays optimally, making it impossible to beat!

---

## 🎮 Features

* 🧠 AI opponent using Minimax algorithm
* 🎨 Clean and modern UI with responsive design
* ⚡ Fast gameplay with smooth interactions
* 🔄 Restart game anytime
* ❌⭕ Human vs AI gameplay

---

## 🕹️ How to Play

* You are **X**, and the AI is **O**
* Click on any empty cell to make your move
* The AI will automatically respond
* The game ends when:

  * You win 🎉
  * AI wins 🤖
  * It's a draw 😐

---

## 🧠 Algorithm Used

The AI uses the **Minimax Algorithm**, a decision-making algorithm commonly used in game theory.

* Evaluates all possible moves
* Chooses the optimal move for the AI
* Ensures the AI never loses
